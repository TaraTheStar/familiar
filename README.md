<h1 align="center">familiar</h1>

A self-hosted, local-network voice companion for the [StackChan](https://github.com/m5stack/StackChan)
ESP32-S3 robot. No cloud: the little robot on your desk wakes when called, hears
you, thinks, and speaks back — all on hardware you own.

The name is the witch's-companion kind: a *familiar* is a spirit bound to a
person. This repo holds the two halves of ours.

**📖 Full user guide:** [QUICKSTART](docs/QUICKSTART.md) in-repo, or the
[docs site](docs/) — quickstart, day-to-day usage, and deep dives on both
halves. The site is Hugo-built; preview it locally with
`hugo server --source docs`.

## The familiars

Say *"be a cat"* — or bat, toad, fox — and the face changes, persisting across
reboots. These are the actual sprite skins the firmware renders on the
320×240 display, driven by the same blink/speaking animation system as the
default face:

| *"be a cat"* | *"be a bat"* |

procedural `default` face ships too.)

## Layout

```
familiar/
├─ poppet/      # the firmware — the little body that sits on your desk
│               #   (M5Stack StackChan app on a vendored xiaozhi-esp32 fork,
│               #    committed in-tree at poppet/vendor/stackchan-esp32/)
├─ grimoire/    # the backend — hears (whisper), thinks (LLM), speaks (Kokoro)
│               #   Go server, deployed as a container
└─ docs/        # project docs — also a GitHub Pages site (Hugo)
   ├─ _index.md            # user-guide landing page (site home)
   ├─ QUICKSTART.md        # clean clone → a talking familiar
   ├─ using.md             # day-to-day guide: wake word, commands, familiars
   ├─ poppet.md            # deep dive: the firmware (the body)
   ├─ grimoire.md          # deep dive: the server (the mind)
   ├─ PROTOCOL_V2.md       # the live v2 contract both halves implement
   ├─ WAKE_WORD.md         # training + flashing the custom wake word
   ├─ sources.md           # lineage — the ancestor projects familiar descends from
   └─ archive/
      └─ PROTOCOL_V1.md    # the retired v1 wire protocol (historical)
```

- **[poppet/](poppet/)** — firmware. Build with ESP-IDF v5.5.4; run
  `python3 fetch_repos.py` first to pull the component libraries (mooncake,
  smooth_ui_toolkit, ArduinoJson, esp-now). See [poppet/BUILD.md](poppet/BUILD.md).
- **[grimoire/](grimoire/)** — Go backend. `make build` (needs the whisper.cpp
  submodule). See [grimoire/README.md](grimoire/README.md).

## First clone

```
git clone https://github.com/TaraTheStar/familiar.git
cd familiar
git submodule update --init --recursive   # pulls whisper.cpp for grimoire
```

## Origins

Both halves were extracted from earlier standalone repos (`StackChan` and
`go-stackend`) as a fresh-start snapshot; their pre-monorepo history lives in
those archives.
